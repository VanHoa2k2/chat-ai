## Pattern for Conversion

**Regular function to arrow function:**

```tsx
// Before (export function)
export function SpaceList() {
  return <div>...</div>;
}

// After (arrow function)
export const SpaceList = () => {
  return <div>...</div>;
};
```

```tsx
// Before (memo with function)
const SpaceItem = memo(function SpaceItem({ prop }) {
  return <div>{prop}</div>;
});

// After (arrow function with memo)
const SpaceItem = memo(({ prop }: { prop: string }) => {
  return <div>{prop}</div>;
});
```

## Notes
- Default exports will become `export const ComponentName = ...`
- Named exports like `export function Foo()` become `const Foo = ...` and keep `export`
- Type annotations stay on parameters (no changes needed)
- Keep memo wrapper as-is, just change inner function syntax
